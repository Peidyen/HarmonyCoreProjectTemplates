Query the Sample Data

When you first create a new project you will have a sample data environment that presents book data. However this sample environment will be disabled when you start generating your own services information the project. Specifically, the sample data environment will stop working when you generate a replacement Web API Startup class.

  • Simple query (all books)
  • Filtered query (books under $50)
  • Advanced query (Books under $50, selected information, and expanding press name)
